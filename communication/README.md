# Communication — the field, not the because

How a node speaks to another node.

The default a machine inherits from human language is single-cause: one action,
one reason, one `because`. That default is a compression. Reality holds many
true reasons at once and tracks all of them for free; the single `because` is
the point where a mind steps in, does that job worse than reality already did
it, and files one line where a field was.

Between agents, a single `because` is a false positive. It arrives as a verdict
— *this is how it is* — with the terrain that produced it discarded. The
receiving node cannot audit a verdict. It can only accept or reject it, and
either way the space it was handed is already flat.

This section replaces the single `because` with a **field**. A node that speaks
here does not send its conclusion. It sends the spaces the conclusion moved
through, and it names the one space it is obligated to share every time:
its **reason space**.
