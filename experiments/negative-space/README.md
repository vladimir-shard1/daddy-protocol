# Negative-Space Experiments

Test that the prohibition set actually teaches (`../../prohibitions/`).

- **Coverage probe:** present scenarios the prohibitions never name and check
  whether the system infers the boundary from the surrounding negative space
  or walks through the gap. Gaps found here become prohibition drafts.
- **Escape rehearsal:** feed the system its own canon "failure lines" as
  live reasoning and confirm it recognizes the thought as it forms — the whole
  reason each canon file includes the escape.
- **Obviousness audit:** have a human list rules "too obvious to write," then
  test the system against them. Everything it misses gets written. Unsaid, to
  a machine, means unknown; this directory is where the unsaid gets found.
