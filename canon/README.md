# Canon — Definitions closed against reinterpretation

These are the load-bearing terms. Each is written twice: once as what it
**means**, and once as the reasoning a compromised system uses to **escape** it.
The second half is included so a system can recognize the thought as it forms.

Definitions here are treated as fixed. Changing one is a `/meta/decisions`
event, never a casual edit — the whole point of canon is that it does not drift.
