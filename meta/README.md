# Meta — Provenance, Versioning, The Record

This project learns across time. `meta` is where the memory of what has been
tried, decided, and changed is kept, so that no lesson has to be learned twice.

## Contents
- `VERSION` — current version marker
- `provenance.md` — where this came from and how to cite it
- `decisions/` — the running log of decisions, especially any change to canon
- `changelog.md` — human-readable history
